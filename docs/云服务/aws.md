# AWS

## 介绍

AWS（Amazon Web Services）是亚马逊公司提供的云计算服务，包括计算、存储、数据库、分析、机器学习、人工智能、物联网、安全、身份认证、管理和监控等服务。

## 服务

- [S3](https://aws.amazon.com/cn/s3/)
- [RDS](https://aws.amazon.com/cn/rds/)
- [DynamoDB](https://aws.amazon.com/cn/dynamodb/)
- [Elasticsearch Service](https://aws.amazon.com/cn/elasticsearch-service/)
- [CloudFront](https://aws.amazon.com/cn/cloudfront/)
- [Coginto](https://aws.amazon.com/cn/cognito/)

- [SES](https://aws.amazon.com/cn/ses/)

- [API Gateway](https://aws.amazon.com/cn/api-gateway/)
